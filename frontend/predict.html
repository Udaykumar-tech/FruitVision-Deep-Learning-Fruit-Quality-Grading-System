How to read these results
  • • Confidence shows how sure each model is. Higher percentage = more certain.
  • • Top prediction is the model's best guess. If > 75% confident, only top-1 is shown. Otherwise top-2 also appears.
  • • "Object Not Detected" means confidence fell below 35% — the model is too uncertain to commit.
  • • Grad-CAM heatmaps show where each model looked. Red = high attention, blue = ignored.
  • • When models disagree, trust the one with higher focus lift (MobileNetV4 > DBA-ViNet > EffNet+SwinT).
